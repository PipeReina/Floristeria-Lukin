Funcionalidades

Gestión nominal

Registra, crea, actualiza y consulta toda la informacion que tiene que ver con la liquidación nominal. Diseñado para las necesidades de la floristeria lukin implementando la nomina Timmpay para el personal

Gestión de Datos

Se tiene en cuenta la informacion total del empleado para poder tener un orden en la documentacion del proceso nominal, siendo concretos a la hora de codificar

Proceso liquidación

El desarrollo nominal se ajusta a todo el personal de la empresa, para calcular el salario se tiene en cuenta datos personales del empleado, cargo, convenio, incapacidad, comision, horas extra, devengado y por ultimo la generación del pago.

Acceso

El administrador consulta la nomina, el tesorero hace el proceso de liquidacion, el contador se encarga de gestionar el gran total y por ultimo el empleado consultara la nomina y hace el debido retiro salarial.
